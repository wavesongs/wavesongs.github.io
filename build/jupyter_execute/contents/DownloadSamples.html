üì• Download Samples#

At present, the most popular websites to download field recordings are Xeno Canto and eBird. The main difference is the data adquisition. To download data from eBird you have to request the audios, Requesting and downloading media, while the datos in Xeno Canto are public and can be donwload via the website or via API, Application Programming Interface (API v2).

Google Colab Execution

If you are using Google Colab install the pacakge via pypi with the following code:

!pip install wavesongs

üé∂ Xeno Canto#

You can query and download audio recordings from Xeno-Canto using the wavesongs.data.xc module. Although this is the fastest way, you can also check the website to get more information about the audios, such as sonograms or recording details. The website also offers a map to browse by region, or you can check by taxonomy.

üîé Query Audios#

First, create a query for the desired sounds. To do this, you need the scientific name of the specie. You can also include additional details such as the country or location where the recording was made to refine your search. Check the Xeno Canto API to check other possible filters.

You can also query multiple species by providing a list of common names along with their corresponding scientific names.

specie_names = [
  ["Rufous-collared Sparrow", "Eurasian Three-toed"],
  ["Zonotrichia capensis", "Picoides tridactylus"]
]

from wavesongs.data.xc import download_audios, query_audios

# Define the query parameters
info = {
  "cnt": "Colombia",
  "loc": "Bogota",
}

df_query = query_audios(
    specie_names = ["Rufous-collared Sparrow", "Zonotrichia capensis"],
    info = info,
    max_nb_files=None, # Number of files to download
    format_time=True,
    format_date=True  
)

‚¨áÔ∏è Download Audios#

Define the download directory (rootdir) for the audio files selected in the query using the dataset parameters. If no dataset name is specified, the default name will be the scientific name, with spaces replaced by underscores (_).

# Download the first 5 files
no_files = 5
df_query_short = df_query.iloc[:no_files]

df_audios = download_audios(
    df_query_short,
    rootdir = "./assets/audio", 
    dataset_name = '',
    overwrite = True,
    save_csv = True,
    verbose = True
)

üé§ Own Records#

Field recordings are highly valuable samples. However, we encourage you to record your own samples using your smartphone. You can use any recording app available in the app store. The recommended sample rate is 44.100 Hz for optimal audio quality.
